Adds real-time solar production data from the SolarEdge Monitoring API. Runs alongside Victron data sources.

Samsung SmartThings Optional

Adds SmartThings devices (switches, power meters, sensors) to the dashboard. Get a Personal Access Token from account.smartthings.com/tokens.

Homey Optional

Connects to Athom Homey Pro 2023+ or Homey Cloud. All devices (lights, switches, sensors, thermostats, locks) are auto-discovered and appear in the dashboard with full control.

Somfy TaHoma Optional

Connects to a Somfy TaHoma or Connexoon box on your local network. Roller shutters, screens, venetian blinds, awnings, gates, and garage doors are auto-discovered and controllable from the dashboard.

Roborock Optional

Roborock-app vacuums (e.g. Q Revo) connect via the Roborock cloud with your account email + password. Older Xiaomi-paired robots use a local IP + miio token instead.

Denon AVR Optional

Controls Denon (and Marantz) AV receivers over the local network via the Telnet control protocol (port 23). Supports power on/standby, master volume, mute, and input selection.

Sonos Optional

Controls Sonos speakers on your local network via UPnP (port 1400). Auto-discovers all Zone Players via SSDP, or specify IPs manually. Provides play/pause, prev/next track, volume, and mute. No account required.

AuxAir / AC Freedom Optional

Controls AUX air conditioners via the AC Freedom cloud API. Supports on/off, temperature, mode (cool/heat/dry/fan/auto), and fan speed. Requires an AC Freedom account.

Suppla Optional

Connects to the Suppla cloud (cloud.supla.org) or a self-hosted Suppla server. Discovers all channels (switches, dimmers, thermometers, humidity sensors, shutters, gates) and groups them by device. Authentication uses a personal access token from the Suppla Cloud → Security → Personal Access Tokens page.

Bayrol Pool Manager Connect Optional

Connects to the Bayrol Pool Access cloud to read pH, ORP (redox), temperature, and free chlorine from a Pool Manager Connect device. Polling interval is 60 s by default.

Dreame Optional

Connects to Dreame robot vacuums and air purifiers locally via the Mi Home (miio) protocol. Each device needs its local IP and 32-char hex token — enable Developer mode in Mi Home, go to the device detail page, tap the three-dot menu → Share Device → 拷贝到剪贴板 to see the token, or use a third-party token extractor.

Satel INTEGRA Optional

Connect to a Satel INTEGRA alarm panel via ETHM-1 Plus module (TCP port 7094). Zone sensors (violation + tamper), partition arm state (arm/disarm control), and programmable output relays appear as individual dashboard devices.

UniFi Protect Optional

Auto-discovers cameras (shown in dashboard) and sensors (door, motion, temperature, humidity) from a UniFi Dream Machine or UNVR.

Loxone Miniserver Optional

Connects to a Loxone Miniserver via WebSocket. All rooms, switches, dimmers, blinds, thermostats, and sensors are auto-discovered from the structure file.

Loxone Outbound Push Optional

Pushes sensor values from LSH to Loxone Virtual Inputs in real time (on change, debounced 200 ms). Configure one mapping per line: storeKey = VirtualInputName. Example: bayrol/19048/ph = PoolPH

Loxone XML Templates

Generates ready-to-import Virtual Output (commands) and Virtual HTTP Input (feedback) templates for Loxone Config 17.1 from your connected devices. Filter by brand, pick the API token to embed, and download — then import via Virtual Outputs / Virtual HTTP Inputs → template import in Loxone Config.

📞 SIP / Intercom Optional

Register as a SIP extension on UniFi Talk, Loxone, or any SIP PBX with WebSocket transport. Incoming calls (e.g. from a UniFi Access door reader) pop up with the matching camera feed and a one-tap unlock button.

LG ThinQ Optional

Connects to your LG ThinQ account via the v1 cloud API. Use your LG account email and password to fetch tokens and user number automatically, or paste them manually. Password is never stored.

IKEA Dirigera Optional

Connects to an IKEA Dirigera hub (2022+). Discovers lights, outlets, blinds, motion sensors, and VINDSTYRKA air quality sensors. Requires a bearer token obtained once via node scripts/dirigera-auth.js <hub-ip> — press the button on the hub when prompted.

IKEA Tradfri Optional

Connects to an older IKEA Tradfri gateway via CoAP/DTLS. Requires npm install node-tradfri-client. First run: enter the security code from the sticker on the gateway — generated identity and PSK are printed to the server log. Subsequent runs: leave the code blank and fill in identity + PSK.

BoneIO Optional

Connects to a BoneIO controller via MQTT. Relays, inputs, and sensors are auto-discovered through Home Assistant MQTT discovery. Leave host blank to use the Cerbo GX MQTT broker.

SmartBob Optional

Connects to SmartBob via MQTT. Add the broker address and define each entity with its state topic, optional command topic, and type.

Entities

Arduino / Generic MQTT Optional

Subscribe to any MQTT topic and map JSON fields to sensor readings or controllable switches. Works with Arduino, ESP32, Tasmota custom firmware, or any device publishing JSON over MQTT.

Sensor type: omit for read-only · "toggle" for on/off switch · "range" for slider (add "min"/"max").
Override per-sensor with "stateTopic" / "commandTopic". Use "jsonKey" if the JSON key differs from "path".

ESPHome Optional

Connects to ESP32/ESP8266 devices running ESPHome with the web_server: component enabled. Sensors, switches, binary sensors, lights, and covers are auto-discovered via the device event stream.

Shelly Optional

Add Shelly Gen1 and Gen2 smart devices by IP address. Switches, dimmers, plugs, and sensors are auto-detected and polled every 30 s.

BroadLink RM4 IR/RF Optional

Control BroadLink RM4 Pro / RM4 Mini IR blasters and RF radio transmitters. Learn IR and RF codes by name, store them locally, and trigger them from the dashboard or via API. MAC address is printed on the device label.

Code Library

Fibaro Home Center Optional

Connects to a Fibaro Home Center 2 or Home Center 3 (Lite) via its local REST API. Devices are grouped by room into cards on the dashboard. Switches, dimmers, roller shutters, and sensors are discovered automatically.

Waveshare Modbus TCP Optional

Control Waveshare relay boards over Modbus TCP (port 502). Supports any Waveshare relay module with Ethernet or a serial-to-Ethernet converter. Each board appears as a device card with individual relay toggles.

Aeotec 360 Camera Optional

Add an Aeotec Indoor Camera 360° by local IP. Sensor data (motion, sound) flows via SmartThings; the local RTSP stream is added to your Cameras list for live video. Credentials are the ones you set in the Aeotec app.

Reolink Optional

Reolink PoE cameras and NVRs. One row per camera — use channel 0 for a standalone camera, or add a row per NVR channel. Snapshots are proxied so the password stays server-side.
