SolarEdge Optional

Adds real-time solar production data from the SolarEdge Monitoring API. Runs alongside Victron data sources.

Samsung SmartThings Optional

Adds SmartThings devices (switches, power meters, sensors) to the dashboard. Get a Personal Access Token from account.smartthings.com/tokens.

Homey Optional

Connects to Athom Homey Pro 2023+ or Homey Cloud. All devices (lights, switches, sensors, thermostats, locks) are auto-discovered and appear in the dashboard with full control.

Dreame Optional

Connects to Dreame robot vacuums and air purifiers locally via the Mi Home (miio) protocol. Each device needs its local IP and 32-char hex token — enable Developer mode in Mi Home, go to the device detail page, tap the three-dot menu → Share Device → 拷贝到剪贴板 to see the token, or use a third-party token extractor.

Satel INTEGRA Optional

Connect to a Satel INTEGRA alarm panel via ETHM-1 Plus module (TCP port 7094). Zones and partitions appear as dashboard sensors.

UniFi Protect Optional

Auto-discovers cameras (shown in dashboard) and sensors (door, motion, temperature, humidity) from a UniFi Dream Machine or UNVR.

Loxone Miniserver Optional

Connects to a Loxone Miniserver via WebSocket. All rooms, switches, dimmers, blinds, thermostats, and sensors are auto-discovered from the structure file.

📞 SIP / Intercom Optional

Register as a SIP extension on UniFi Talk, Loxone, or any SIP PBX with WebSocket transport. Incoming calls (e.g. from a UniFi Access door reader) pop up with the matching camera feed and a one-tap unlock button.

IKEA Dirigera Optional

Connects to an IKEA Dirigera hub (2022+). Discovers lights, outlets, blinds, motion sensors, and VINDSTYRKA air quality sensors. Requires a bearer token obtained once via node scripts/dirigera-auth.js <hub-ip> — press the button on the hub when prompted.

IKEA Tradfri Optional

Connects to an older IKEA Tradfri gateway via CoAP/DTLS. Requires npm install node-tradfri-client. First run: enter the security code from the sticker on the gateway — generated identity and PSK are printed to the server log. Subsequent runs: leave the code blank and fill in identity + PSK.

BoneIO Optional

Connects to a BoneIO controller via MQTT. Relays, inputs, and sensors are auto-discovered through Home Assistant MQTT discovery. Leave host blank to use the Cerbo GX MQTT broker.

Shelly Optional

Add Shelly Gen1 and Gen2 smart devices by IP address. Switches, dimmers, plugs, and sensors are auto-detected and polled every 30 s.

Aeotec 360 Camera Optional

Add an Aeotec Indoor Camera 360° by local IP. Sensor data (motion, sound) flows via SmartThings; the local RTSP stream is added to your Cameras list for live video. Credentials are the ones you set in the Aeotec app.
